### Draw pixels from scratch

#### Project to learn how to draw pixels and shapes only using pure C

#### Quick Start to see my amazing canvas

`git clone git@github.com:toxypiks/draw_pixels_from_scratch.git`\
`cd draw_pixels_from_scratch`\
`mkdir build`\
`cd build`\
`cmake ..`\
`make`\
`./main`

* Install feh

`feh output.ppm &`
